# Sistema de Gestión de Inventario
## Descripción

Aplicación desarrollada en Python para administrar un inventario de productos electrónicos.

El sistema permite registrar, buscar, modificar, eliminar y listar productos, además de generar estadísticas y almacenar la información de forma permanente utilizando archivos JSON.

---

## Funcionalidades

- Registrar productos.
- Listar productos.
- Buscar productos por código.
- Buscar productos por categoría.
- Modificar productos.
- Eliminar productos.
- Mostrar estadísticas del inventario.
- Guardar y cargar información automáticamente desde un archivo JSON.
